Data description for 2024_WaterQuality_FlowExp.zip

This collection consists of a series of data sets comprising the environmental parameter measured along the First 12 weeks of experiments and their exploration using R


Description of data files:
	2024_FlowExp_Waterquality_data_sheet: a .xlsx file including Dissolved oxygen, temperature, pH and Salinity measurements over the first 7 weeks of the experiments among the 3 flumes
	2024_WaterQuality_DataAnalysis: a .Rmd file made for data exploration
	Environmental_Parameter_Over_Time: .png file showing the evolution of the environmental parameters over the 12 weeks
